# ⏱️ ChronoSpeak

ChronoSpeak is a simple and interactive web project built using **HTML, CSS, and JavaScript**.  
It allows users to **convert speech into text** in real time using the browser’s Speech Recognition API.

---

## 🚀 Features

- 🎤 **Speech to Text Conversion**  
- ⚡ Real-time recognition and display  
- 🌐 Works directly in the browser  
- 🖥️ Simple and beautiful UI  
- 🔊 Start/Stop voice recognition buttons  
- 📄 Copy or use the generated text anywhere

---

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript**  
- **Web Speech API (SpeechRecognition)**

## ⚙️ How to Run the Project

1. Download or clone the repository  
2. Open the folder in VS Code  
3. Right-click on **index.html**  
4. Click **“Open with Live Server”**  
   (or simply open it in your browser)
